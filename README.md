# ![](icon.png "") ガゾーナラベ by 片山博文MZ

「ガゾーナラベ」は、複数の画像を並べて Word 文書ファイル（docx）にするソフトです。

基本操作は、画像ファイルをリストに追加して「docx生成」ボタンを押すだけです。

完成したdocxファイルはデスクトップに作成されます。

## 対応環境

日本語 Windows 10
